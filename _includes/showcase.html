Simple to use

First, install Edge Benchmark from PyPi. You can also install Edge Benchmark Client from source but we recommend the installation using pip. 
pip install edge_benchmark

Next, create a model and send it for benchmarking on real devices. You can select what devices you want to benchmark on. Edge Benchmark also support more detailed settings, like number of used CPUs, GPU use, number of threads and others. For more information, you can browse through documentation for Edge Benchmark Client. 
from edge_benchmark import Benchmark
user_token = "demo_token"
tflite_path = "./model.tflite"
b = Benchmark()
b.all_devices()
b.set_token(user_token)
measurements = b.measure(tflite_path)
print(measurements)

Finally, client returns measurements results in JSON format. Measurements contain both global and layer-wise statistics executed on selected devices. We recommend to use our Edge Benchmark Platform where you can easily see and compare benchmark results of your model.
